## 🚀 Featured Project — AstroBurst

<a href="https://github.com/samuelkriegerbonini-dev/AstroBurst">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=samuelkriegerbonini-dev&repo=AstroBurst&theme=tokyonight&hide_border=true" alt="AstroBurst repository card" />
</a>

**[AstroBurst](https://github.com/samuelkriegerbonini-dev/AstroBurst)** is an open-source desktop app for processing astronomical images. Drop in your **FITS** or **ASDF** files, compose RGB from narrowband channels, stack with sigma clipping, and export — everything runs **100% locally**, with no cloud dependencies.

It's the **first astronomical image processor built on the Rust / Tauri / WebGPU stack**, and the **first non-Python tool with native ASDF support** (Roman Space Telescope). Works with JWST (NIRCam/MIRI), Hubble, and Roman data.

![Rust](https://img.shields.io/badge/Rust-000000?style=for-the-badge&logo=rust&logoColor=white)
![Tauri](https://img.shields.io/badge/Tauri-24C8DB?style=for-the-badge&logo=tauri&logoColor=black)
![React](https://img.shields.io/badge/React_19-61DAFB?style=for-the-badge&logo=react&logoColor=black)
![WebGPU](https://img.shields.io/badge/WebGPU-005A9C?style=for-the-badge&logo=webgpu&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)

**⚡ Performance**
- Opens a **2 GB IFU datacube in 300 ms**
- Processes **10 frames at 1.4 GB/s**
- Renders STF adjustments in **8 ms on GPU** (4K)

**✨ Highlights**
- **10-step ComposeWizard** pipeline: channels → stack → align → crop → background → blend → color → stretch → adjust → export
- **GPU RGB preview** via WebGPU — pixel-identical STF across GPU shader, CPU worker, and Rust backend
- **Sub-pixel alignment** (Foroosh phase correlation + star-based affine with RANSAC)
- **SPCC** color calibration against real **Gaia DR3**, interactive photometry, and plate solving
